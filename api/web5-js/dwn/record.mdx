## Overview
Every modifying method of the [Records](records.mdx) class returns an instance of a `Record` object, which is a representation of the record(s) involved in the operation

## Methods

### send(did)

Sends the record to the DWN endpoints of a provided DID

#### Parameters

<ApiDetails
  details={[
    {
      data: {
        field: 'did',
        value: 'The DID associated with the DWN endpoints where the record will be sent',
      }
    }
  ]}
/>


#### Return Value

The response object of the request

#### Code Example
```js
const { record } = await web5.dwn.records.create({
  data: "Hello World!",
  message: {
    dataFormat: "text/plain",
  },
});

//highlight-start
const { status } = await record.send("did:example:bob");
//highlight-end
```

<Divider type="slash"/>

### delete()

Deletes record from DWN

#### Return Value

The response object of the request

#### Code Example
```js
const response = await record.delete();
```
